# Название файла: .md — Центр анализа входящих файлов

`analysis_hub` — временная рабочая зона для входящих файлов на анализ.

## Структура

- `analysis_hub/inbox/` — сюда ты складываешь Excel/CSV/JSON/DOCX/TXT для разбора.
- Источник правды хранится в `docs/backend-map/`:
  - `small_dictionaries.json`
  - `entities_from_dictionaries.json`

## Протокол обработки

1. Я читаю файлы из `inbox/`.
2. Анализирую и сравниваю со схемой в `docs/backend-map/`.
3. Обновляю JSON в `docs/backend-map/` новыми полями/справочниками без дублей.
4. Удаляю из `inbox/` только файлы, которые полностью разобраны.
5. Если файл не разобран полностью — оставляю его в `inbox/` и пишу: `Этот документ еще не проанализирован`.
