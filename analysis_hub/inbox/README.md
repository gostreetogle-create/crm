# Название файла: .md — Входящая папка для файлов на анализ

Клади сюда любые входные файлы для разбора:
- `.xlsx`, `.xls`, `.csv`
- `.json`, `.txt`, `.md`
- документы с описанием полей/таблиц

После обработки:
- разобранные файлы удаляются из этой папки;
- неразобранные остаются до следующего прохода.
