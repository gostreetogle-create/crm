# Surface Finishes CRUD

Справочник `Финиш/шероховатость` хранит единый набор параметров поверхности для материалов.

## Поля

- `finishType` — тип финиша (`Glossy`, `Matte`, `Satin`, `Semi-gloss` и т.д.)
- `roughnessClass` — класс/обозначение шероховатости (`Ra 0.8`, `N7`, ... )
- `raMicron` — числовой параметр `Ra` (мкм), опционально

## Интеграция с материалами

- В форме материала есть dropdown выбора записи из этого справочника.
- Кнопка `+` открывает quick-add модалку.
- При выборе справочного значения поля `finishType/roughnessClass/raMicron` подставляются в материал автоматически.
