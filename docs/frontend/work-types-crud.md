# Вид работ (manufacturing) — CRUD в `/dictionaries`

Справочник операций для маршрутов и планирования. Нормы **времени** по конкретной детали/заказу задаются в маршруте, не в строке справочника; в карточке вида работ хранится **учётная ставка труда (руб/ч)** для расчёта стоимости: `часы × hourlyRateRub` (часы — из операции маршрута). См. `crm-manufacturing-dictionaries.json` и `docs/business-logic-canon.md`.

Единый паттерн полей и режимов UI: [`dictionary-field-behavior-guide.md`](./dictionary-field-behavior-guide.md).

## Поля (домен ↔ UI)

Стабильные имена в коде и Excel — ниже. **Подписи:** в таблице короткие; в модалке create/edit/view — полные. См. `docs/frontend/dictionaries-naming-convention.md`.

| JSON / домен          | TS              | Колонка таблицы (`label`) | Модалка (`label`)     |
| --------------------- | --------------- | ------------------------- | --------------------- |
| — (системный ключ)   | `id`            | —                         | —                     |
| Наименование         | `name`          | Наименование              | Наименование          |
| Короткое_обозначение | `shortLabel`    | Коротк. обозн.            | Короткое обозначение  |
| Ставка_руб_час       | `hourlyRateRub` | ₽/ч                       | Ставка, ₽/ч           |
| активность           | `isActive`      | Активна                   | Активна               |

**Наименование:** уникально в UX (сравнение без учёта регистра); на бэке — UNIQUE на `name` (или эквивалент). Отдельного бизнес-кода нет; FK — по **`id`**.

**Ставка ₽/ч:** целое число рублей за час по этому виду работ; в Excel колонка `Ставка руб/ч`. **Минимум 1 ₽/ч** — сохранение и импорт без нуля и без пустой ставки. Себестоимость труда по строке маршрута: норма/факт часов с операции × ставка из справочника (при согласовании — со снимком на заказ/КП по общим правилам репозитория).

Отдельного поля «тип операции» / технологической группы **нет**: смысл процесса задаётся **наименованием** (например «Лазерная резка», «Сварка», «Порошковая покраска») и коротким обозначением.

## Excel

Шаблон и импорт: колонки `Наименование`, `Короткое обозначение`, `Ставка руб/ч`, `Активна` (Да/Нет). Импорт — только создание (`createMany`); дубликаты наименований в файле и пересечение со справочником — отказ с сообщением по строке.
