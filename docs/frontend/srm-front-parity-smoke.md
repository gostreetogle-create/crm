# Parity: `crm-web` vs `srm-front` (smoke)

Проверять после изменений в общих libs (`@srm/*`).

| Сценарий | Ожидание |
|----------|----------|
| Вход mock dev (`admin`/`admin`) | Редирект на `/справочники`, шапка и тема |
| Хаб справочников | Плитки, раскрытие, CRUD без консольных ошибок |
| Выход | Сессия сброшена, форма логина |
| Несуществующий URL в `srm-front` | Страница 404 |
| Страница `forbidden` | Текст 403, ссылка на главную |

Команды: `npx nx serve srm-front` и `npx nx serve crm-web` (тот же backend/proxy при необходимости).
