# КП A4 — чеклист входных файлов

Напоминание перед сборкой шаблонов КП в фирменном стиле.

## Минимум (чтобы стартовать)

- [ ] Передан `PDF` эталонного бланка КП (в финальном виде для печати)

## Рекомендуется (чтобы сделать без потерь по стилю)

- [ ] Передан исходник `PSD` (слои, сетка, направляющие)
- [ ] Переданы превью страниц `PNG/JPG` (для быстрой визуальной сверки)
- [ ] Передан список шрифтов (семейство + начертания)
- [ ] Переданы фирменные цвета (`HEX`/`CMYK`) и логотипы (`SVG/PNG`)

## Параметры печати (уточнить до верстки)

- [ ] Формат: `A4`
- [ ] Поля страницы
- [ ] Вылеты (`bleed`, обычно 3 мм) — нужны/не нужны
- [ ] Цветность: полноцвет / вариант для ч/б печати

## Definition of Ready

- [ ] PDF получен
- [ ] Требования печати подтверждены
- [ ] Можно собирать универсальный шаблон КП под CRM
