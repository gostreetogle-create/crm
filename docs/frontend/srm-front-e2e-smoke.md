# E2E smoke для `srm-front` (минимум)

Автоматизацию Playwright/Cypress можно подключить отдельно. Ручной чеклист:

1. Открыть `/`, ввести учётные данные, убедиться в навигации на `/справочники`.
2. Открыть произвольный несуществующий путь — заголовок 404.
3. После входа открыть `/forbidden` — текст недоступности (или сценарий с `permissionGuard`).
4. В хабе: одна модалка создания записи без 500 в сети.

CI: после установки Playwright достаточно одного spec на пункты 1–2.
