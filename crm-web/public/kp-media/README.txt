Фото для справочника «Фото для КП» (и для Excel).

Кладите файлы в эту папку репозитория:
  crm-web/public/kp-media/

В CRM / Excel в поле «URL фото» указывайте относительный путь от корня сайта:
  /kp-media/имя-файла.jpg

Локально (ng serve):  http://localhost:4200/kp-media/имя-файла.jpg
На сервере:           https://ваш-домен/kp-media/имя-файла.jpg

Файлы попадают в сборку Angular как статика; отдельный каталог на сервере / в Docker не нужен.
