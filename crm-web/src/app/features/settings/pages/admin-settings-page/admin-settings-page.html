Матрица разрешений для текущего браузера: страницы, плитки на хабе «Справочники», кнопки в карточках и Excel. Роли — из справочника «Роли». Суперадминистратор в таблице не показывается: у него всегда полный доступ, матрица на него не действует. Для остальных ролей по умолчанию права пустые, пока вы не отметите галочки; «К канону» снимает сохранённые переопределения и возвращает к этому пустому умолчанию. Порядок колонок совпадает с полем Порядок в справочнике «Роли» (меньше число — левее). Матрица хранится в localStorage до сброса.

Проверить интерфейс «как под другой ролью» в этом браузере — в Настройках (шестерёнка). Назначить роль сотруднику — в «Справочниках» → «Пользователи». Ключ страницы: page.admin.settings.
